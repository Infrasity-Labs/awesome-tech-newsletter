## Contributing

Contributions are very welcome! Please ensure that the newsletter you're adding is active and provides value to developers. 

1. Fork this repository.
2. Add your newsletter to the appropriate section following the same table structure: `| Name | Link | Description | Frequency |`.
3. Submit a pull request!
